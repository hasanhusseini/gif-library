# Privacy

GIF Library is a local-first Windows desktop app for saving and organizing GIFs, images, and remote media links.

## Local Data

GIF Library stores library data locally on your computer. Local files imported into the app may be copied into app-managed local storage.

Exports may contain remote URLs, notes, tags, aliases, folder names, and, with a full backup, copies of local files.

## Network Behavior

GIF Library does not upload files to cloud storage.

Remote media links may be previewed or fetched from their original hosts. Those requests come from your computer and can expose your IP address to those hosts.

## Discord And Tracking

GIF Library does not:

- connect to Discord
- use Discord tokens
- automate Discord
- scrape private APIs
- include telemetry
- run as a background service
