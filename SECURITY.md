# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.0 private/public source release | Yes |

## Reporting A Vulnerability

Please report security issues privately to the project maintainer. Do not post exploit details, proof-of-concept code, or sensitive reproduction steps in public issues, discussions, pull requests, or social channels.

When reporting, include:

- affected version
- operating system
- a short description of the issue
- steps to reproduce, if they can be shared safely
- any relevant logs or screenshots with private data removed

GIF Library is a local-first desktop app. Reports involving unexpected network access, unsafe local file handling, import/export issues, or privacy-impacting behavior are especially useful.
